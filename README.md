# Techno Alarm (Кибер-Будильник)

Стильный будильник в стиле Cyberpunk Modern на Jetpack Compose.

## Особенности
- **Cyberpunk Aesthetic**: Неоновые свечения, глубокий черный фон, моноширинные шрифты.
- **Навязчивый режим**: Будильник не прекратит играть, пока вы не нажмете кнопку "STOP ALARM" на весь экран.
- **Работа поверх блокировки**: Экран включается автоматически при срабатывании.
- **Системные часы**: Отображение времени системы прямо в приложении для точной настройки.

## Технологии и версии
Для корректной работы проекта на другом ПК убедитесь, что у вас установлены соответствующие версии:

- **JDK**: 17 или выше (рекомендуется для AGP 8.4.0)
- **Android Studio**: Jellyfish (2023.3.1) или новее
- **Gradle**: 9.2.0 (включен в проект через wrapper)
- **Android Gradle Plugin (AGP)**: 8.4.0
- **Kotlin**: 2.0.0
- **Compose Compiler**: 2.0.0 (интегрирован в плагин Kotlin)
- **Min SDK**: 24 (Android 7.0)
- **Target/Compile SDK**: 34 (Android 14)

## Как запустить на другом ПК

1. **Клонирование**:
   ```bash
   git clone <url-вашего-репозитория>
   cd android_alarm_clock
   ```

2. **Открытие в Android Studio**:
   - Выберите `File -> Open` и укажите корневую папку проекта.
   - Дождитесь завершения синхронизации Gradle.

3. **Настройка SDK**:
   - Убедитесь, что установлен SDK Platform 34 (Android 14).
   - В настройках проекта (`File -> Project Structure -> SDK Location`) проверьте, что используется JDK 17+.

4. **Запуск**:
   - Подключите устройство или запустите эмулятор (API 24+).
   - Нажмите `Run 'app'`.

## Важные нюансы
- **Разрешения**: При первом запуске приложение запросит разрешение на уведомления (на Android 13+). Также для работы точных будильников может потребоваться подтверждение в системных настройках (приложение само перенаправит вас туда при необходимости).
- **Режим 24ч**: Будильник работает строго в 24-часовом формате (например, 21:00 вместо 09:00 PM).
- **Звук**: Если на эмуляторе нет звука, проверьте системную громкость будильника в настройках Android (Settings -> Sound).

## Структура проекта
- `MainActivity.kt`: Основной интерфейс и логика планирования.
- `AlarmReceiver.kt`: Принимает сигнал от системы и запускает экран/сервис.
- `AlarmService.kt`: Управляет цикличным воспроизведением звука.
- `AlarmRingingActivity.kt`: Экран срабатывания будильника с кнопкой отключения.
